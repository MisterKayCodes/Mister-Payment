# Mister Payment

Telegram payment confirmation bot (Aiogram v3)

## Features

- Dynamic payment methods (admin configurable)  
- Users upload payment screenshots  
- Admin approves/declines payments  
- JSON-based storage (lightweight, easy to manage)  
- Fully async and commands-first  
- Admin contact editable anytime  

## Requirements

- Python 3.11+  
- Aiogram v3  
- Install dependencies: `pip install -r requirements.txt`

## Setup

1. Copy `.env.example` to `.env` and fill your BOT_TOKEN, admin ID, etc.  
2. Ensure `storage/` and `logs/` folders exist  
3. Run bot: `python main.py`
